Snake Solver

GitHub Video demo

Snake Solver is a Python game written using the Pygame library during Jolly's Game Jam. The game is similar to snake except you have to eat the number which solves the equation at the top in order to advance to the next level. The game is 2-player to fit Jolly's Game Jam theme of "Co-op".

The video shows how control of the snake switches between players every 3rd level as depicted by a color change of the snake. This allows each player to trade-off roles of focusing on solving the equation and navigating the snake safely.
