Intern at MSFC

I was an intern for the Center Chief Technologist at NASA'S Marshall Space Flight Center in Huntsville, Alabama. I worked on building a physical display highlighting the Center's significant technology achievements. In addition to this project, I worked on a side project helping to integrate RCS code for Nanolauncher into a pub/sub framework setup written in C++ and wrapped in lua.

Snake Solver

Snake Solver is a Python game written using the Pygame library during Jolly's Game Jam. The game is similar to snake except you have to eat the number which solves the equation at the top in order to advance to the next level.

Stock Feed

This smartwatch app provides stock and cryptocurrency information to the user. It was developed during HackTX with my group of friends. We used a Samsung Galaxy Gear, Bloomberg's API, and a Microsoft Azure server in order to get the functionality of the app just right. My job was writing the backend using Bloomberg's API to retrieve stock information.
